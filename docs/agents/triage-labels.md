# Triage labels

The `triage` skill applies these labels as it processes an issue through its state machine.

| Label | Meaning |
|-------|---------|
| `needs-triage` | A maintainer needs to evaluate this issue |
| `needs-info` | Waiting on the reporter to provide more detail |
| `ready-for-agent` | Fully specified — an AI coding agent can pick this up with no additional human context |
| `ready-for-human` | Needs a human to implement |
| `wontfix` | Will not be actioned |
