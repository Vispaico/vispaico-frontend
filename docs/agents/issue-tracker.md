# Issue tracker

Issues for this repo live in **GitHub Issues** on `https://github.com/Vispaico/vispaico-frontend`.

## Tooling

The `gh` CLI must be installed and authenticated. Agent skills will use:

- `gh issue create` — to open new issues
- `gh issue list` — to query existing issues
- `gh issue view` — to read a specific issue
- `gh issue edit` — to update labels, assignees, milestones
